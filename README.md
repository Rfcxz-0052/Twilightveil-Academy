# 暮影學院 Twilightveil Academy

一款瀏覽器視覺小說遊戲。

## 🎮 遊戲特色
- 雙世界系統（表世界 / 裡世界）
- 好感度系統
- 五行靈力設定
- 節點式劇情分歧

## 🌌 世界觀
在普通的大學校園背後，隱藏著只有覺醒者才能看見的「靈學院」。
學生分為滅鬼派與渡鬼派，而主角將在理念衝突中做出選擇。

## 🛠 使用技術
- HTML
- CSS
- JavaScript
